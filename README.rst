Install
------------------------------------------------------------------------------

``junzheng_demo`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install junzheng-demo

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade junzheng-demo
